# Expense Tracker 💰

A CLI-based personal expense tracker built with Core Java.

## Features
- Add new expenses
- View all expenses
- Calculate total spending
- Data persists using file storage (CSV)

## Technologies Used
- Java
- Maven
- File I/O
- Git and GitHub

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run Main.java
